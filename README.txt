This plugin generate a sitemap for home page inernal links
== Description ==

Following task performed by the plugin
1. Generate Sitemap for home page 
2. Display sitemap to admin
3. Display sitemap.html file to front end user
4. Regenerate sitemap in every 1 hr
== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `home-sitemap` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the setting section and you will find the sitemap plugin there
